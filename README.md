## Restaurant Menu.

## 📌 | 🇺🇸 Description | 🇧🇷 Descrição  
 - 🇺🇸 | A restaurant menu developed for Micro Frontend studies using Next.js. Users can add or remove meals from the cart and view the total price.
 - 🇧🇷 | Cardápio online para estudo de Micro Frontend utilizando Next.js. Usuários podem adicionar ou excluir refeições no carrinho e verificar o total.

## 🚀 | 🇺🇸 How to run the project locally | 🇧🇷 Instruções para rodar localmente
  > - 🇺🇸 | 1. Download the .zip file from the GitHub repository (https://github.com/Clique-N/Restaurant-Menu-Micro-Frontend).
  > - 🇧🇷 | 1. Faça o download do arquivo .zip no repositório do GitHub (https://github.com/Clique-N/Restaurant-Menu-Micro-Frontend).
  
  > - 🇺🇸 | 2. Extract the .zip file to a folder your computer, then open the project in VS Code.
  > - 🇧🇷 | 2. Extraia o arquivo .zip na pasta do seu computador, e em seguida abra no VS Code.

  > - 🇺🇸 | 3. Open three terminals, one for each Micro Frontend.
  > - 🇧🇷 | 3. Abra três terminais, um para cada Micro Frontend.

  > - 🇺🇸 | 4. In each terminal, run the following command to install the dependencies: `npm install`.
  > - 🇧🇷 | 4. Em cada terminal, execute o seguinte comando para instalar as dependências: `npm install`

  > - 🇺🇸 | 5. After installing the dependencies, run the following command in each terminal to start the applications locally: `npm run dev`.
  > - 🇧🇷 | 5. Após instalar as dependências, execute o seguinte comando em cada terminal para iniciar as aplicações localmente: `npm run dev`.

  > - 🇺🇸 | 6. After starting all Micro Frontend applications, open the `container` application in your browser using the localhost URL provided in the terminal.
  > - 🇧🇷 | 6. Após iniciar todas as aplicações Micro Frontend, acesse a aplicação `container` no navegador usando a URL localhost exibida no terminal.
